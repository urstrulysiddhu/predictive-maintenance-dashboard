# Predictive Maintenance Analytics Dashboard

An interactive Tableau dashboard analyzing industrial machine sensor data to identify potential failure patterns.

## Project Overview

This project visualizes predictive maintenance data to help detect patterns that may indicate machine failures before they occur.

## Key Insights

- Highlights failure distribution across machine components
- Shows patterns in operational conditions leading to failures
- Helps identify high-risk equipment conditions

## Tools Used

- Tableau Public
- Data Visualization
- Industrial Data Analysis
